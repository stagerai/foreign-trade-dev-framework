# 客户假设验证框架

**方法论来源:** 学术项目假设检验框架迁移
**适用范围:** 客户画像验证、邮件策略验证、回复率分析

---

## 一、假设验证流程（四阶段）

| 阶段 | 操作 | 输出 | 时间 |
|------|------|------|------|
| **阶段1：提出假设** | 定义假设内容 | 假设卡片 | 5分钟 |
| **阶段2：设计验证** | 确定验证方法和数据 | 验证计划 | 10分钟 |
| **阶段3：收集数据** | 从实际操作中收集数据 | 数据记录 | 按周期 |
| **阶段4：得出结论** | 分析数据得出结论 | 验证结论 | 10分钟 |

---

## 二、假设类型分类

### 客户画像类假设（H00X_P）

| 假设类型 | 示例 | 验证方法 |
|---------|------|---------|
| **公司类型假设** | "经销商回复率高于集成商" | 发送邮件后统计回复率 |
| **规模假设** | "20-200人公司回复率最高" | 发送邮件后统计回复率 |
| **行业假设** | "安防行业比IT行业回复率高" | 发送邮件后统计回复率 |
| **区域假设** | "拉美回复率高于东南亚" | 发送邮件后统计回复率 |

### 邮件策略类假设（H00X_S）

| 假设类型 | 示例 | 验证方法 |
|---------|------|---------|
| **主题行假设** | "具体主题行回复率高于泛泛主题" | A/B测试不同主题行 |
| **篇幅假设** | "50词邮件回复率高于100词" | A/B测试不同篇幅 |
| **CTA假设** | "低门槛CTA回复率高于高门槛" | A/B测试不同CTA |
| **轮次假设** | "第3轮回复率最高" | 统计各轮次回复率 |
| **角度假设** | "案例角度回复率高于极简角度" | A/B测试不同角度 |

### 时间类假设（H00X_T）

| 假设类型 | 示例 | 验证方法 |
|---------|------|---------|
| **发送时间假设** | "周二上午发送回复率最高" | 不同时间发送统计 |
| **间隔假设** | "10天间隔回复率最高" | 不同间隔统计 |

## 三、假设卡片格式

```markdown
## 假设卡片

**假设ID：** H001_P
**假设类型：** 客户画像类
**提出日期：** YYYY-MM-DD

### 假设内容
**假设：** [假设描述]

### 验证方法
**方法：** [验证方法描述]
**数据来源：** 开发日志回复记录
**样本量要求：** ≥[N]封邮件

### 验证数据

| 数据项 | 组A | 组B |
|--------|-----|-----|
| 发送邮件数 | | |
| 回复邮件数 | | |
| 回复率 | % | % |

### 验证结论
**结论日期：** YYYY-MM-DD
**结论：** □ 假设成立 □ 假设不成立 □ 待验证
**置信度：** 高/中/低
```

## 四、假设验证周期

| 假设类型 | 验证周期 | 数据量要求 |
|---------|---------|-----------|
| 客户画像类 | 发送50+封后验证 | ≥50封（每类≥25） |
| 邮件策略类 | A/B测试20+封后验证 | ≥20封/组 |
| 时间类 | 发送30+封后验证 | ≥30封 |

## 六、假设验证总表维护

**更新频率：**
- 每次提出新假设：新增假设行
- 每次验证假设：更新验证状态和结论
- 每周复盘：检查待验证假设进度
