# 邮件模板库使用指南

**用途：** 各轮邮件的标准模板，起草时参考

---

## 模板文件清单

| 文件 | 轮次 | 篇幅 | 策略 | CTA |
|------|------|------|------|------|
| `第1轮_极简介绍.md` | 第1轮 | 30-50词 | 一句话价值主张 | Interested? |
| `第2轮_案例证明.md` | 第2轮 | 50-70词 | 同类客户案例 | Want details? |
| `第3轮_供应商角度.md` | 第3轮 | 40-60词 | 提及现有供应商 | Want to compare? |
| `第4轮_破冰询问.md` | 第4轮 | 30-40词 | Should I stop? | Yes or No |
| `第5轮+_轻问候.md` | 第5轮+ | 30-50词 | 行业动态，不推销 | Stay in touch |

---

## 使用流程

### 步骤1：确定轮次

**读取开发日志，确认：**
- 当前是第几轮？
- 回复状态（有回复 vs 沉默）
- 已用主题/角度（避免重复）

### 步骤2：打开对应模板

| 轮次 | 打开文件 |
|------|---------|
| 第1轮 | `第1轮_极简介绍.md` |
| 第2轮 | `第2轮_案例证明.md` |
| 第3轮 | `第3轮_供应商角度.md` |
| 第4轮 | `第4轮_破冰询问.md` |
| 第5轮+ | `第5轮+_轻问候.md` |

### 步骤3：参考模板起草

1. 参考正文结构模板
2. 参考合格示例
3. 填入客户具体信息
4. 检查毅冰铁律是否遵守

---

## 重要提醒

> "模板是框架，不是直接复制。每个客户不同，每轮角度不同。"
> "第2轮和第1轮说一样的话，客户直接删掉。"

- 参考模板结构
- 填入客户具体信息
- 不直接复制模板原文
- 每轮主题行与历史不重复
- 每轮角度与历史不重复

---

## 模板选择对照表

| 情况 | 轮次 | 选择模板 |
|------|------|---------|
| 初次联系 | 第1轮 | `第1轮_极简介绍.md` |
| 第1轮沉默后 | 第2轮 | `第2轮_案例证明.md` |
| 第2轮沉默后 | 第3轮 | `第3轮_供应商角度.md` |
| 第3轮沉默后 | 第4轮 | `第4轮_破冰询问.md` |
| 第4轮沉默后 | 第5轮+ | `第5轮+_轻问候.md` |
| 有回复 | 任意轮 | 不用模板，针对回复回应 |
