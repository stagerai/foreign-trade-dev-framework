# 第2轮邮件模板：案例证明

**适用轮次：** 第2轮（第1轮沉默7天后）
**篇幅要求：** 50-70词
**策略核心：** 用同类客户成功案例证明价值
**CTA类型：** Want details?

---

## 毅冰铁律（必须遵守）

| 铁律 | 说明 |
|------|------|
| **用案例证明** | "用案例证明你不是空口说白话" |
| **案例必须具体** | 城市 + 客户类型 + 具体结果 |
| **与第1轮角度不同** | 第1轮是价值主张，第2轮是案例证明 |
| **承认间隔** | "Following up" / "Following up on my earlier email" |

---

## 主题行设计

| 要素 | 示例 |
|------|------|
| **案例城市** | [City] [industry] case |
| **客户类型** | How a [industry] distributor won more projects |
| **具体数字** | 40% fewer callbacks case |

---

## 正文结构

```
主题：[案例城市] + [客户类型] case

Hi [姓名],

Following up. [同类客户案例，含城市、类型、结果]。[具体产品型号/规格]。

Want details?

Best regards,
[Your Name] | [YourCo]
```

## 评分标准（第2轮）

| 维度 | 满分 | 第2轮要求 |
|------|------|---------|
| 主题行具体性 | 20 | 包含案例城市/客户类型 |
| 主题行长度 | 10 | ≤10词 |
| 正文篇幅 | 15 | ≤70词 |
| 差异化 | 20 | **与第1轮主题/角度完全不同** |
| 价值具体性 | 15 | 含具体案例（城市+类型+结果） |
| CTA门槛 | 10 | Want details? / Want the case study? |
| 联系人适配 | 10 | 适合联系人角色 |
