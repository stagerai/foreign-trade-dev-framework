# 第3轮邮件模板：供应商角度

**适用轮次：** 第3轮（第2轮沉默后）
**篇幅要求：** 40-60词
**策略核心：** 提及客户现有供应商，说"多一个选择"，不说"替代"
**CTA类型：** Want to compare?

---

## 毅冰铁律（必须遵守）

| 铁律 | 说明 |
|------|------|
| **不说"替代"** | "不要说'替代现有供应商'，要说'多一个选择'" |
| **提及现有供应商** | 降低信任门槛，客户看到熟悉的名字 |
| **提及价格优势** | 不说"便宜"，说"更好的定价" |
| **与第1-2轮角度不同** | 前两轮是价值/案例，第3轮是供应商角度 |

---

## 主题行设计

| 合格主题 | 说明 |
|---------|------|
| Another [来源] supplier (like [现有供应商]) | 直接提及现有供应商 |
| Second source for [产品类型] | 第二供应商角度 |
| Backup option for [公司名]'s [产品] | 备份供应商角度 |

---

## 正文结构

```
主题：Another [来源] supplier (like [现有供应商])

Hi [姓名],

I see [公司名] works with [现有供应商]. We're another [来源] manufacturer - same grade, better pricing. [差异化优势1-2句]。

Want to compare pricing?

Best regards,
[Your Name] | [YourCo]
```

---

## 合格模板示例

### 示例1：同为中国供应商

```
主题：Another China supplier (like [Existing Brand])

Hi John,

I see ExampleCo works with [Existing Brand] (China). We're another Chinese manufacturer - same industrial grade, better pricing. ERPS <15ms, PoE++ 96W, 5-year warranty.

Want to compare pricing?

Best regards,
[Your Name] | [YourCo]
```

### 示例2：备份供应商

```
主题：Backup switch supplier for ExampleCo

Hi Jane,

No response needed if this isn't relevant. But if ExampleCo ever needs a backup switch supplier — same specs as [Existing Brand], factory pricing, 2-4 week delivery.

Want to compare?

Best regards,
[Your Name] | [YourCo]
```

## 评分标准（第3轮）

| 维度 | 满分 | 第3轮要求 |
|------|------|---------|
| 主题行具体性 | 20 | 包含"另一个供应商"角度 |
| 主题行长度 | 10 | ≤10词 |
| 正文篇幅 | 15 | **≤60词** |
| 差异化 | 20 | **与第1-2轮主题/角度完全不同** |
| 价值具体性 | 15 | 含差异化优势（价格/交期/服务） |
| CTA门槛 | 10 | Want to compare? |
| 联系人适配 | 10 | 适合联系人角色 |

---

## 第3轮检查清单

```
□ 主题行提及"另一个供应商"或"备份"
□ 主题行与第1-2轮不重复（必须）
□ 正文 ≤60 词
□ 正文提及客户现有供应商
□ 不说"替代"
□ CTA为 Want to compare?
□ 与第1-2轮角度完全不同
```
