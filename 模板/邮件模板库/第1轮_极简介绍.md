# 第1轮邮件模板：极简介绍

**适用轮次：** 第1轮（初次联系）
**篇幅要求：** 30-50词（毅冰铁律：客户只给你10秒）
**策略核心：** 一句话价值主张，让客户看完就知道你要干什么
**CTA类型：** Interested?

---

## 毅冰铁律（必须遵守）

| 铁律 | 说明 |
|------|------|
| **不说"我们是制造商"** | 客户不在乎你是谁，只在乎你能帮他解决什么问题 |
| **主题行必须具体** | 提及客户业务/痛点，不说"合作机会" |
| **正文≤50词** | 客户只给你10秒，看完就知道你要干什么 |
| **CTA最低门槛** | Interested?（1个单词可回复） |

---

## 主题行设计

**主题行四要素（必须包含至少1个）：**

| 要素 | 说明 | 示例 |
|------|------|------|
| **客户业务** | 提及客户产品/行业 | [Industry] reliability for [Company] |
| **客户痛点** | 提及客户可能的问题 | Remote project callbacks? |
| **具体数字** | 提及价值数字 | 40% fewer callbacks |
| **客户城市** | 提及客户所在地 | [City] distributors |

**禁止主题行：** Partnership opportunity / About [YourCo] / Best industrial products

---

## 正文结构（毅冰四句法）

```
主题：[客户业务] + [痛点/数字] for [公司名]

Hi [姓名],

[客户业务关联句，1句]。[单一价值主张，1-2句，含具体数字]。

Interested?

Best regards,
[Your Name] | [YourCo]
```

## 合格模板示例

### 示例1：安防批发商

```
主题：CCTV reliability for ExampleCo's remote projects

Hi John,

ExampleCo's nationwide security deployments need reliable networking. Our industrial switches (96W/port, ERPS <15ms) help distributors reduce callbacks 40%.

Interested?

Best regards,
[Your Name] | [YourCo]
```

**词数：** 42词
**主题要素：** 客户业务（CCTV）+ 痛点（reliability）+ 数字（40%）
**CTA：** Interested?

## 评分标准（第1轮）

| 维度 | 满分 | 第1轮要求 |
|------|------|---------|
| 主题行具体性 | 20 | 包含客户业务/痛点/数字 |
| 主题行长度 | 10 | ≤10词 |
| 正文篇幅 | 15 | ≤50词 |
| 差异化 | 20 | （第1轮无历史，直接满分） |
| 价值具体性 | 15 | 含具体数字/规格 |
| CTA门槛 | 10 | Interested? / Want specs? |
| 联系人适配 | 10 | 适合联系人角色 |

---

## 第1轮检查清单

```
□ 主题行包含客户业务/痛点/数字（至少1个）
□ 主题行 ≤10 词
□ 正文 ≤50 词
□ 正文含具体数字/规格
□ CTA为 Interested? 或 Want specs?
□ 不说"我们是制造商"
□ 不说"合作机会"
□ 不用 Free/Best/Discount
```
