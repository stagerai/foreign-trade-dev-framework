## 重写触发检查（毅冰红线）

**以下红线触发则必须重写：**

| 红线 | 检查 | 结果 |
|------|------|------|
| □ 主题行泛泛（合作/机会/Partnership） | □ 是 → **必须重写** | |
| □ 主题行过长（>10词） | □ 是 → **必须重写** | |
| □ 正文超限（>要求20%） | □ 是 → **必须重写** | |
| □ 主题行与历史重复 | □ 是 → **必须重写** | |
| □ 角度与历史重复 | □ 是 → **必须重写** | |
| □ CTA高门槛 | □ 是 → **必须重写** | |
| □ 无具体元素（数字/案例） | □ 是 → **必须重写** | |

---

## 操作决定

□ **通过 → 提交Critic评审**（得分≥80，所有维度≥门槛）
□ **针对性优化 → 修复后重新评分**（某维度<门槛）
□ **进入修复循环**（得分70-79）
□ **必须重写**（得分<70 或 红线触发）
